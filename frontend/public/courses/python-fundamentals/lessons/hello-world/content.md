# Hello, World!

Every programmer's journey starts here. You're going to write a program that prints a message to the screen.

## What you'll learn

- How to use `print()` to display text
- How strings work in Python (text inside quotes)
- How to run a Python program

## Instructions

1. Look at the starter code in `main.py` — it prints `"Hello, Python!"`.
2. Click **Run** to see the output.
3. Change the message so it prints exactly: `Hello, World!`
4. Click **Run** again. When the output matches, the lesson is complete.

## Key concept

`print()` is a built-in function that sends text to the output. The text you want to print goes inside the parentheses, wrapped in quotes:

```python
print("your message here")
```

You can use either single quotes `'...'` or double quotes `"..."` — both work the same way.

## Hints

- Make sure the output matches exactly, including the comma and exclamation mark.
- Python is case-sensitive: `Print` is not the same as `print`.
